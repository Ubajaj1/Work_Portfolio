# Personal Portfolio - AI Engineer & Creative

A minimalistic, premium portfolio designed for an AI Engineer with passions for photography and fitness. Built with pure HTML, CSS, and JavaScript for maximum performance and zero dependencies.

## 🚀 Features

- **Premium Dark Mode**: Sleek, modern aesthetic with glassmorphism effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Interactive Elements**: Custom cursor, smooth scroll animations, and hover effects.
- **Three Pillars**: dedicated sections for Engineering (AI), Creativity (Photography), and Discipline (Fitness).
- **Zero Dependencies**: No frameworks, just pure web technologies.

## 🛠 Setup & Customization

1.  **Open the project**: Simply open `index.html` in your browser to view the site.
2.  **Edit Content**:
    *   Open `index.html` in your code editor.
    *   Update the text in the **Hero**, **About**, and **Projects** sections with your own details.
    *   Update links in the **Contact** section.
3.  **Add Images**:
    *   Place your project images in `assets/images/`.
    *   Update the CSS classes in `css/style.css` (look for `.project-image` or `.photo-item`) to point to your actual images instead of the gradients.
    *   Example:
        ```css
        .p1 {
            background-image: url('../assets/images/your-photo.jpg');
        }
        ```

## 📦 Deployment (Free)

You can host this for free using **GitHub Pages**:

1.  Create a new repository on GitHub.
2.  Push this code to the repository.
3.  Go to **Settings** > **Pages**.
4.  Select the `main` branch as the source.
5.  Your site will be live at `https://yourusername.github.io/repo-name`.
